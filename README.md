# IntroGym

**IntroGym** - это интуитивно понятное и функциональное фитнес-приложение для iOS. С **IntroGym** вы сможете легко создавать персонализированные тренировки и отслеживать свой прогресс, делая ваши фитнес-цели более достижимыми

## Основные возможности
- **Тренировки:** создавайте и редактируйте тренировки, добавляя разнообразные упражнения
- **Планирование:** запланируйте свои тренировки на любой удобный день
- **Упражнения:** выполняйте и редактируйте подходы для каждой тренировки
- **Справочник:** подробная информация об упражнениях, разделенная по группам мышц
- **Статистика:** мотивирующий раздел
- **Профиль:** вся информация о текущем пользователе в одном месте

## Технологии
- **Язык программирования:** Swift
- **Среда разработки:** Xcode
- **Фреймворки:** UIKit (Programmatic)
- **Хранение данных:** CoreData
- **СУБД:** SQLite

## Установка
1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/username/FitnessApp.git
    ```
2. Перейдите в папку проекта:
    ```bash
    cd FitnessApp
    ```
3. Откройте проект в Xcode:
    ```bash
    open FitnessApp.xcodeproj
    ```
4. Запустите проект на симуляторе или реальном устройстве.

## Планируемые улучшения
- Редизайн интерфейса для улучшения пользовательского опыта
- Анимированная визуализация упражнений в справочнике для наглядного выполнения
- Расширение справочника с увеличением количества упражнений
- Создание интерактивных виджетов для быстрого доступа к основным функциям или отображения статистики
- Интеграция системы уведомлений для напоминаний и мотивации
- Введение системы наград и достижений для повышения мотивации
- Встроенный таймер для отслеживания времени выполнения тренировки
- Удобный поиск упражнений в справочнике
- Добавление графиков и диаграмм для визуализации пользовательской статистики
- Включение калькуляторов для подсчета калорий, белков, жиров и углеводов, а также идеального веса и других параметров
- Возможность записи уровня тяжести каждого подхода
- Интерактивное обучение при первом запуске приложения для новых пользователей
- Локализация приложения на английский язык
